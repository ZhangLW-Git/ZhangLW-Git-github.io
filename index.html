分析英语文本最多单词数

本代码是按照小白的生活逻辑思维，若使用“字典”思想会更加方便。
不多说，上代码：


text = open("abc.txt").readlines()
    lineall = ""
for line in text:
    lineall += line #输出信息

lettertable = []
for i in range(65,91):
    lettertable.append(chr(i))
    j = i-65+97
    lettertable.append(chr(j))
lettertable.append(["'", "-"])     #字母表

words = []
word = ""
for letter in lineall:
    if letter in lettertable:
        word += letter
    else :
        if word != "":
            words.append(word)
            word = ""              #存储单词

k = 0
for i in words:
    for j in words:
        if words.count(i) >= words.count(j):
            k += 1
        if k == len(words):
            print(i, words.count(i)) #记数


上述代码可提取同目录下“abc.txt”内的消息，并计算出现最多的单词，并记数。学习python最好的是自己多思考多敲击代码。
